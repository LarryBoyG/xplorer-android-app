# Xplorer Android App

Open source Android app development for XIRO Xplorer drone compatibility and protocol research.

## Overview

This repository contains ongoing Android app development intended to restore or replace important mobile functionality for the XIRO Xplorer drone ecosystem.

Current areas of work include:

- RTSP video support
- Camera control research
- Telemetry and protocol investigation
- Range extender and Wi-Fi behavior analysis
- Clean-room Android compatibility development

## Status

Active development. Experimental and incomplete.
